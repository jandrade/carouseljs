Slider

A lightweight and flexible carousel/slider. It is a javascript component built using raw javascript, to avoid library dependencies. The component supports css3 transitions, infinite navigation, responsive behaviors and other stuff.

Options

  • infinite: Enables infinite/circular navigation (default: false)
  • forceTouch: Forces dragging behavior (default: false)
  • auto: Enables automatic sliding (default: false)
  • duration: Duration time between slides on auto enabled (default: 5000)
  • vertical: Enables vertical navigation (default: false)
  • breakpoints: Enables vertical navigation (default: false)

Examples

Default


                var slider = new bite.Slider('.slider-default');

Single


                var sliderSingle = new bite.Slider('.slider-single', {
                    forceTouch: true,
                    single: true
                });

Auto


                var sliderAuto = new bite.Slider('.slider-auto', {
                    auto: true,
                    single: true,
                    forceTouch: true,
                    duration: 4000
                });

Without infinite scroll


                var sliderNoInf = new bite.Slider('.slider-no-inf', {
                    infinite: false
                });

Vertical Scrolling


                var sliderVertical = new bite.Slider('.slider-vertical', {
                    forceTouch: true,
                    infinite: false,
                    vertical: true,
                    single: true
                });
